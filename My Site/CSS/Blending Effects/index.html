The filter property

- hover over the element to see which filter is applied

CODE: 
// HTML ---------------
    <img id="fxnone" src="Images/lightning.jpg" alt="McQueen" title="none">
    <img id="fxblur" src="Images/lightning.jpg" alt="McQueen" title="blur">
    <img id="fxbrightness" src="Images/lightning.jpg" alt="McQueen" title="brightness">
    <img id="fxcontrast" src="Images/lightning.jpg" alt="McQueen" title="contrast">
    <img id="fxgreyscale" src="Images/lightning.jpg" alt="McQueen" title="greyscale">
    <img id="fxhue-rotate" src="Images/lightning.jpg" alt="McQueen" title="hue-rotate">
    <img id="fxinvert" src="Images/lightning.jpg" alt="McQueen" title="invert">
    <img id="fxopacity" src="Images/lightning.jpg" alt="McQueen" title="opacity">
    <img id="fxsaturate" src="Images/lightning.jpg" alt="McQueen" title="saturate">
    <img id="fxsepia" src="Images/lightning.jpg" alt="McQueen" title="sepia">
    <img id="fxdrop-shadow" src="Images/lightning.jpg" alt="McQueen" title="drop-shadow">
    <img id="fxmix" src="Images/lightning.jpg" alt="McQueen" title="fxmix">
    
// CSS ---------------
    #fxnone{                                    #fxblur{                                        #fxbrightness{                              
        filter: none;                               filter: blur(7px);                              filter: brightness(1000%);                  
        -webkit-filter: none;                       -webkit-filter: blur(7px);                      -webkit-filter: brightness(1000%);          
    }                                           }                                               }                                           

    #fxcontrast{                                #fxgreyscale{                                   #fxhue-rotate{
        filter: contrast(500%);                     filter: grayscale(100%);                        filter: hue-rotate(180deg);
        -webkit-filter: contrast(500%);             -webkit-filter: greyscale(100%);                -webkit-filter: hue-rotate(180deg);
    }                                           }                                               }

    #fxinvert{                                  #fxopacity{                                     #fxsaturate{
        filter: invert(100%);                       filter:opacity(25%);                            filter: saturate(500%);
        -webkit-filter: invert(100%);               -webkit-filter:opacity(25%);                    -webkit-filter: saturate(500%);
    }                                           }                                               }

    #fxsepia{                                   #fxdrop-shadow{                                             
        filter: sepia(85%);                         filter: drop-shadow(12px 12px 8px green);
        -webkit-filter: sepia(85%);                 -webkit-filter: drop-shadow(12px 12px 8px green);
    }                                           }

    #fxmix{
        filter: drop-shadow(12px 12px 8px green) blur(2px) hue-rotate(180deg);
        -webkit-filter: drop-shadow(12px 12px 8px green) blur(2px) hue-rotate(180deg);
    }

The filter property with svg filter filter: url()

CODE : 
// HTML ----------------------
    <img id="svgFilterTest" src="Images/lightning.jpg" alt="McQueen" title="SVG filter">
    
    <svg id="svg-filter">
    <filter id="svg-blur">
        <feGaussianBlur in="SourceGraphic" stdDeviation="4"></feGaussianBlur>
    </filter>
    </svg>        
    
// CSS ------------------
    #svgFilterTest {
        max-width: 250px;
        height: auto;
        filter: url(#svg-blur);
    }

isolation property TEST

- the blending mode is set on the grand-child element (element with the image)

- the child element (element with blue border) is where the isolation:isolate; is set (this layer isolates the grand-child element)

square-1: the middle square (blue border) has no background set so the grand-child element has nothing to blend with

square-2: the middle square (blue border) has a green background set so the grand-child element blend with it but not with the parent


- here is how this works the element which has the isolation:isolate; set will isolate all children element form its parent element form any blending but the element intself will still blend with its children elements

CODE : 
-- CODE (HTML) -- 
    <div class="isolate">
        <div class="isolateCh_1">
            <div class="isolateGrandCh_1"></div>
        </div>
        <div class="isolateCh_2">
            <div class="isolateGrandCh_2"></div>
        </div>
    </div>
        
-- CODE (CSS) -- 
    .isolate {                                          // parent element 
        width: 400px;
        height: 200px;
        border: solid 2px black;
        display: flex;
        background-color: red;
    }
    .isolateCh_1 {                                      // isolator element 1
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        isolation: isolate;                             
    }
    .isolateGrandCh_1 {
        margin: auto;
        width: 100px;
        height: 100px;
        border: solid 2px black;
        background-image: url(Images/rock.jpg);
        background-size: cover;
        mix-blend-mode: screen;
    }
    .isolateCh_2 {                                      // isolator element 2
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        background-color: green;
        isolation: isolate;
    }
    .isolateGrandCh_2 {
        margin: auto;
        width: 100px;
        height: 100px;
        border: solid 2px black;
        background-image: url(Images/rock.jpg);
        background-size: cover;
        mix-blend-mode: screen;
    }
