nodemon available on npm

- automatically restarts the server when a change is detected in the server file

- simple to use, instead of node use nodemon (ex: sudo nodemon index)

pyenv python version manager

- allows to switch between python versions (it also allows to use the system python version if we want to)

- global version:

- we can switch between the global (system level) python versions

- each global python version has its own packages separated from each other

- local version

- we can specify which python version to use in a directory (a .python-version folder created in the directory to keep track of this) 

    pyenv                                      // list available commands 
    
    pyenv global ver                           // list the global currently used python version (switches version if ver specified)
    
    pyenv local ver                            // list the local currently used python version (switches version if ver specified)
                                               // creates a .python-version directory locally, in the current directory the python version specified in this directory will be used (not the global version)    
    
    pyenv versions                             // list installed python versions (managed py pyenv)
    pyenv install --list                       // prints available versions to install 
    pyenv install ver                          // installs the specified python version 
    pyenv uninstall ver                        // uninstalls the specified python version

nvm node version manager

- allows to switch between node versions (it also allows to use the system node version if we want to)

- global version:

- we can switch between the global (system level) node versions

- each global node version has its own packages separated from each other

- local version

- we can specify which node version to use in the current shell 

    nvm                                        // lists available nvm commands
    
    nvm use ver                                // locally switches to the specified node version (for the current shell)
    
    nvm alias default ver                      // globally switches to the specified node version  
    
    nvm ls                                     // list installed node versions (managed by nvm)
    nvm ls-remote                              // prints available versions to install   
    
    nvm install ver                            // installs the specified node version (node = latest version)   
    nvm uninstall ver                          // installs the specified node version

mypy static type checker for python 

    mypy file.py                               // checks for typing errors (does not run the program)
