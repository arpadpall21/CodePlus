Command Description
list(nr) - prints the source code (as many lines as it is specified)
cont | c - continue program execution
next | n - step next (does not enter functions)
step | s - step in (step next but enters functions)
out | o - step out (step out the current function -> moves down the call stack)
backtrace | bt - print the current execution frame (call stack)
exec exp - executes an expression
restart - restart the script
kill - kills the script (not debugger exit)
repl - opens the repl

Node.js Debugger

Remember This :

Description and Demonstration :

- Node.js includes a debugging utility accessible via a V8 Inspector, nodejs will stop on the first line if started with debugger

- I'ts recommended to use graphical debuggers, I write a few examples here but debugging in VS code is so much better 

    > node inspect nodeJsFile            // launches nodejs with debugger
    
    const x = 10;
    const y = 11;
    debugger;                            // inserts a breakpoint in the code (ignored if nodejs is not started with debugger) 
    const z = 12;
