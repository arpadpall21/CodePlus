Description and Demonstration :

- Tailwind uses the dark: modifier to automatically apply dark styles (automatically detect OS dark settings) 

    <htmlTag class="dark:tailwStyle" >              // automatically applies tailwStyle if the browser detects dark mode

- we can also enable manual dark mode switch from tailwind.config.js file 

// tailwind.config.js (file) -------------
    
    /** @type {import('tailwindcss').Config} */
    module.exports = {
      darkMode: 'selector',                 // enables manual light|dark mode switch (further config)   
    }
  
// HTML -------------------------
    <html class="dark">     // adding a dark class in html tag enables dark mode (otherwiser ligth mode)
                                  // we control this behaviour from JavaScript

- getting browser's current dark mode status 

    window.matchMedia('(prefers-color-scheme: dark)').matches       // renturns ture if the OS is in dark mode, ortherwiser false
Example :

Auto Dark Mode 

    <div class="h-10 bg-blue-300 dark:bg-blue-700"></div>

Manual Dark Mode 

// tailwind.config.js (file) -------------
    /** @type {import('tailwindcss').Config} */
    module.exports = {
      content: ['./src/**/*.{html,js}'],
      theme: {
        extend: {}
      },
      plugins: [],
      darkMode: 'selector',
    }
    
// HTML -------------
    <div class="h-10 bg-blue-300 dark:bg-blue-700"></div>
    <button type="button" onClick="enableDarkMode()"> Dark Mode </button>
    
    <script>
    function enableDarkMode(e) {                        // add/remove dark class on html tag
      const dakrModeEnalbed = document.documentElement.classList.contains('dark')
      
      if (dakrModeEnalbed) {
        document.documentElement.classList.remove('dark')
      } else {
        document.documentElement.classList.add('dark')
      }
    }
    </script>
